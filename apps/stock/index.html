Flux conseillé

1. Vous ajustez le stock local. 2. Vous chargez une photo d’étagère. 3. Vous indiquez le comptage détecté par zone. 4. Le module compare et vous propose les écarts à valider.

Mode bêta sans RTSP

Pour l’instant, le contrôle caméra fonctionne avec une image importée depuis votre téléphone ou PC. La caméra en direct remplacera cette source plus tard sans changer la logique métier.

Réglages GitHub

Cette version est entièrement statique : HTML, CSS, JS, localStorage. Elle ne modifie pas les fichiers GitHub automatiquement.

Pour publier un état durable, utilisez Exporter puis remplacez plus tard vos JSON de stock avec une version admin ou un worker. Cloudflare n’est volontairement pas intégré ici.
